# 项目说明

这是一个安全巡检平台，用于管理 Linux 服务器和交换机设备。

目标：
- 自动巡检服务器
- 检测开放端口
- 检查安全基线
- 输出风险和整改建议

---

# 技术要求

- 后端：FastAPI
- 数据库：PostgreSQL
- 前端：Next.js
- 设备连接：SSH（Netmiko）
- 调度：APScheduler

---

# 开发规则（非常重要）

1. 所有功能必须分步骤实现，不允许一次写完整个平台
2. 代码必须结构清晰
3. 每个功能必须可以单独运行
4. 不允许写死规则，必须可扩展
5. 不允许明文保存密码
6. 每一步都要能运行

---

# 当前阶段目标（第一阶段）

只做这几个功能：

1. 资产管理
2. SSH连接测试
3. Linux巡检
4. 交换机巡检
5. 端口扫描
6. 基线检查

不做：
- Windows
- 自动攻击
- 自动整改
